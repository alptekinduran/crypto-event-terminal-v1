# iPhone ile kurulum

## 1) GitHub
- Yeni repo aç
- Bu klasörün içindeki tüm dosyaları repo'ya yükle

## 2) Replit import
- Replit uygulamasında Create sekmesi
- Sağ üst `⋮`
- `Import from GitHub`
- Repo seç

## 3) Secrets
Şunları ekle:
- APP_PASSWORD
- SESSION_SECRET
- VAPID_PUBLIC_KEY
- VAPID_PRIVATE_KEY
- VAPID_SUBJECT
- WATCHLIST_JSON
- CUSTOM_PROJECT_SOURCES
- DEEPSEEK_API_KEY (opsiyonel)
- COINGECKO_API_KEY (opsiyonel)

## 4) Run
- `npm install` gerekirse terminalden çalıştır
- Run bas

## 5) Publish
- Replit Publish / Deploy
- çıkan bağlantıyı Safari'de aç
- Paylaş > Ana Ekrana Ekle
- Ana ekrandan açıp giriş yap
- Bildirimleri Aç'a bas
